# V2 Backlog Item Template

Use this shape for new entries:

- `v2-<domain>-NNN` (`proposed`): <clear improvement outcome>.

Examples:

- `v2-rel-007` (`proposed`): Add automatic recovery workflow when webhook queue repeatedly exceeds critical threshold.
- `v2-ui-007` (`proposed`): Add per-page refresh controls and query-time indicators to improve operator confidence.
- `v2-test-006` (`proposed`): Add regression tests for schema/view contract compatibility across two prior migration versions.
